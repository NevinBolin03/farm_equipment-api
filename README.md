# Farm Equipment API

## Overview
This project is a RESTful API for managing farms, equipment, and maintenance records. It was built as a Minimum Viable Product (MVP) to demonstrate backend development skills including database design, CRUD operations, middleware, and testing.

## Technologies Used
- Node.js
- Express.js
- Sequelize
- SQLite
- Jest & Supertest
